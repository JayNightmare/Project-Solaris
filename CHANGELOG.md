# Change Log

All notable changes to the "project-solaris" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.0.3] - 2025-12-31

-   **New Orbital Engine**: Visualized the file system as a strict hierarchical solar system with orbiting animations.
-   **Galaxy Navigation**: Added "Click-to-Fly" focus navigation and "Zoom-to-Reset" functionality.
-   **Speed Control**: Added a "Warp Speed" slider to control animation speed.
-   **Enhanced Visuals**: Improved "Deep Space" warning and Star/Planet styling.

## [0.0.2]

-   Added support for `.gitignore` and `.ignore` files.
-   Excluded `node_modules` and other ignored files from the system visualization.

## [0.0.1]

-   Initial release
